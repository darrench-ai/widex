Suggested Solutions

如何取得 API Key？

✦ Gemini (Google)

  1. 前往 🔗 Google AI Studio
  2. 登入 Google 帳號 → 點擊 「Create API key」
  3. 複製 Key 貼回本頁 → 儲存

◎ ChatGPT / OpenAI

  1. 前往 🔗 OpenAI Platform
  2. 登入帳號 → 點擊 「Create new secret key」
  3. Key 格式：sk-...

◈ Claude / Anthropic

  1. 前往 🔗 Anthropic Console
  2. 登入帳號 → 點擊 「Create Key」
  3. Key 格式：sk-ant-...
🔒 安全提示： 所有 API Key 僅儲存在您瀏覽器的 localStorage 中，不會傳送到其他伺服器。每個提供者的 Key 獨立儲存。
